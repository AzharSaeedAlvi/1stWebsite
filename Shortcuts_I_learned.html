These are actually helpful

  • to close the current tab press ---> ctrl + w
  • to shift along tab in chrome use ctrl+shift+tab or ctrl+tab based on left and right
  • to search use ctrl+L or simply press /
  • to use cursor without using mouse turn on the num key and use num pad to navigate(sidenote- if you want to speed the cursor just press ctrl while using the numpad )
  • to open any taskbar application click on win(super key)+(number at which the application lies on taskbar)
  • to select only a single line just go to that line and press shift+down arrow
  • to open emoji press win+.
